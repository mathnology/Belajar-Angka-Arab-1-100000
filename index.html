📝 Rumus Angka 2 Digit (21-99)

🔢 Pola Umum:

Satuan + و + Puluhan

(Units) + (wa) + (Tens)

📋 Contoh:

25 خَمْسَةٌ وَعِشْرُونَ
37 سَبْعَةٌ وَثَلَاثُونَ
84 أَرْبَعَةٌ وَثَمَانُونَ

💡 Catatan: Kata "و" (wa) berarti "dan" yang menghubungkan satuan dengan puluhan.

📝 Rumus Angka 3 Digit (101-999)

🔢 Pola Umum:

Ratusan + Satuan + و + Puluhan

(Hundreds) + (Units) + (wa) + (Tens)

Khusus untuk ratusan:
  • • 100 = مائة (mi'ah)
  • • 200 = مائتان (mi'atan)
  • • 300-900 = [angka] + مائة

📋 Contoh:

125 مِائَةٌ خَمْسَةٌ وَعِشْرُونَ
237 مِائَتَانِ سَبْعَةٌ وَثَلَاثُونَ
584 خَمْسُمِائَةٍ أَرْبَعَةٌ وَثَمَانُونَ

📝 Rumus Angka 4 Digit (1001-9999)

🔢 Pola Umum:

Ribuan + Ratusan + Satuan + و + Puluhan

(Thousands) + (Hundreds) + (Units) + (wa) + (Tens)

Khusus untuk ribuan:
  • • 1000 = ألف (alf)
  • • 2000 = ألفان (alfain)
  • • 3000-9000 = [angka] + آلاف (alaf)

📋 Contoh:

1234 أَلْفٌ مِائَتَانِ أَرْبَعَةٌ وَثَلَاثُونَ
2567 أَلْفَانِ خَمْسُمِائَةٍ سَبْعَةٌ وَسِتُّونَ
3890 ثَلَاثَةُ آلَافٍ ثَمَانُمِائَةٍ تِسْعُونَ

📝 Rumus Angka 5 Digit (10001-99999)

🔢 Pola Umum:

Puluhan Ribu + Ribuan + Ratusan + Satuan + و + Puluhan

(Ten Thousands) + (Thousands) + (Hundreds) + (Units) + (wa) + (Tens)

Khusus untuk puluhan ribu:
  • • 10000 = عشرة آلاف (asyaratu alaf)
  • • 20000 = عشرون ألفًا (isyrun alf)
  • • 30000-90000 = [puluhan] + ألفًا (alf)

📋 Contoh:

12345 اِثْنَا عَشَرَ أَلْفًا ثَلَاثُمِائَةٍ خَمْسَةٌ وَأَرْبَعُونَ
25678 خَمْسَةٌ وَعِشْرُونَ أَلْفًا سِتُّمِائَةٍ ثَمَانِيَةٌ وَسَبْعُونَ
50000 خَمْسُونَ أَلْفًا

💡 Catatan: Untuk angka puluhan ribu, kata "ألفًا" (alfan) digunakan setelah puluhan, berbeda dengan "آلاف" (alaf) untuk ribuan.

🚀 Panduan Cepat

🔤 Kata Penghubung

  • • و (wa) = "dan"
  • • Menghubungkan satuan-puluhan
  • • Selalu di tengah angka 2 digit

📊 Urutan Penulisan

  • • Ribuan → Ratusan
  • • Satuan → و → Puluhan
  • • Baca dari kanan ke kiri

⚡ Tips Cepat

  • • Hafalkan angka dasar 1-10
  • • Pelajari puluhan 20-90
  • • Gabungkan dengan rumus
